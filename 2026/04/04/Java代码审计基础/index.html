дҝ®еӨҚж–№жЎҲ

  1. еҹәдәҺеҗҺз«ҜйҳІеҫЎпјҢдҪҝвҪӨй…ҚзҪ®йЎ№вҪ¬зҡ„е…ЁеұҖиҝҮж»ӨеҷЁпјҢе°Ҷиҫ“вјҠзҡ„зү№ж®Ҡеӯ—з¬ҰиҪ¬д№үеҗҺеӯҳеӮЁеҲ°ж•°жҚ®еә“пјҢжҲ–иҖ…иҝҮж»ӨжҺүзү№ж®Ҡеӯ—з¬ҰпјҢеҸӘдҝқз•ҷжӯЈеёёж•°жҚ®пјӣ

  2. еҹәдәҺеүҚз«ҜйҳІеҫЎпјҢдҪҝвҪӨCSPгҖҒHttpOnlyгҖҒеүҚз«Ҝзү№ж®Ҡеӯ—з¬ҰиҪ¬д№үзӯүвјҝж®өиҝӣвҫҸйҳІеҫЎгҖӮ

е®Ўи®ЎжҠҖе·§

е…ЁеұҖжҗңзҙўж•°жҚ®еә“зҡ„жҸ’е…ҘиҜӯеҸҘ(е…ій”®иҜҚпјҡinsert,save,update)пјҢз„¶еҗҺжүҫеҲ°иҜҘжҸ’е…ҘиҜӯеҸҘжүҖеұһзҡ„ж–№жі•еҗҚеҰӮ(insertUser())пјҢз„¶еҗҺе…ЁеұҖжҗңзҙўиҜҘж–№жі•еңЁе“ӘйҮҢиў«и°ғз”ЁпјҢдёҖеұӮеұӮзҡ„и·ҹиёӘгҖӮзӣҙеҲ°жүҫеҲ°getParamter()ж–№жі•иҺ·еҸ–иҜ·жұӮеҸӮж•°зҡ„ең°ж–№еҒңжӯўпјҢеҰӮжһңжІЎжңүе…ЁеұҖ XSS иҝҮж»ӨеҷЁпјҢи·ҹиёӘзҡ„ж•ҙдёӘжөҒзЁӢйғҪжІЎжңүеҜ№иҺ·еҸ–зҡ„еҸӮж•°иҝҮж»ӨпјҢеҲҷеӯҳеңЁеӯҳеӮЁеһӢ XSSгҖӮ

жҲ–иҖ…д»ҺgetParamter()ж–№жі•жүҫпјҢи·ҹиёӘиҜ·жұӮеҸӮж•°пјҢзӣҙеҲ°жҸ’е…Ҙж•°жҚ®еә“зҡ„иҜӯеҸҘпјҢеҰӮжһңдёӯй—ҙжІЎжңүиҝҮж»ӨеҸӮж•°пјҢеҲҷеӯҳеңЁеӯҳеӮЁеһӢ XSSгҖӮ

вҪӮд»¶дёҠдј жјҸжҙһпјҢжҳҜжҢҮвҪӨжҲ·дёҠдј дәҶвјҖдёӘеҸҜжү§вҫҸзҡ„и„ҡжң¬вҪӮд»¶(еҰӮjsp/php/asp)пјҢе№¶йҖҡиҝҮжӯӨи„ҡжң¬вҪӮд»¶иҺ·еҫ—дәҶжү§вҫҸжңҚеҠЎеҷЁз«Ҝе‘Ҫд»Өзҡ„иғҪвј’гҖӮеёёв»…еңәжҷҜжҳҜwebжңҚеҠЎеҷЁе…Ғи®ёвҪӨжҲ·дёҠдј еӣҫвҪҡжҲ–иҖ…жҷ®йҖҡвҪӮжң¬вҪӮд»¶дҝқеӯҳ

йҳІжҠӨжүӢж®ө

дё»иҰҒйҖҡиҝҮйҷҗеҲ¶ж–Үд»¶зұ»еһӢгҖҒйҮҚе‘ҪеҗҚж–Үд»¶гҖҒжЈҖжҹҘж–Үд»¶еҶ…е®№иҝҷеҮ дёӘжүӢж®өжқҘйҳІжҠӨ

е®Ўи®ЎжҠҖе·§

еңЁжҹҘзңӢж—¶пјҢдё»иҰҒеҲӨж–ӯжҳҜеҗҰжңүжЈҖжҹҘеҗҺзјҖеҗҚпјҢеҗҢж—¶иҰҒжҹҘзңӢй…ҚзҪ®ж–Үд»¶жҳҜеҗҰжңүи®ҫзҪ®зҷҪеҗҚеҚ•жҲ–иҖ…й»‘еҗҚеҚ•

е…ЁеұҖжҗңзҙўд»ҘдёӢзҡ„вҪӮд»¶ж“ҚдҪңе…ій”®еӯ—пјҢи·ҹиҝӣеҸҜжҺ§еҸҳйҮҸпјҡ

йҒҮеҲ°е…Ғи®ёдёҠдј zipж–Үд»¶ж—¶еҸҜд»ҘеҜ№еә”зңӢзңӢжңүжІЎжңүе®ҡд№үиҮӘи§ЈеҺӢпјҲUnzipпјүж–№жі•пјҢеҰӮжһңеӯҳеңЁиҮӘи§ЈеҺӢдё”дёҚеҜ№еҺӢзј©ж–Үд»¶еҒҡж ЎйӘҢпјҢеҸҜд»ҘйҖҡиҝҮжһ„йҖ zipеҢ…жқҘиҝӣиЎҢд»»ж„Ҹж–Үд»¶дёҠдј 

еҸҰеӨ–иҝҳеҸҜд»Ҙе…іжіЁuploadPathжңүжІЎжңүиў«еҶҷжӯ»пјҢеҰӮжһңеҸҜжҺ§жҳҜеҗҰиғҪеҶҷе…Ҙ../иҝӣиЎҢзӣ®еҪ•з©ҝи¶ҠдёҠдј еҲ°жҢҮе®ҡзӣ®еҪ•

XXE

XXE (XML External Entity Injection), XMLеӨ–йғЁе®һдҪ“жіЁвјҠпјҢеҪ“ејҖеҸ‘вјҲе‘ҳй…ҚзҪ®е…¶XMLи§ЈжһҗеҠҹиғҪе…Ғи®ёеӨ–йғЁе®һдҪ“еј•вҪӨж—¶пјҢж”»еҮ»иҖ…еҸҜеҲ©вҪӨиҝҷвјҖеҸҜеј•еҸ‘е®үе…Ёй—®йўҳзҡ„й…ҚзҪ®вҪ…ејҸпјҢе®һж–Ҫд»»ж„ҸвҪӮд»¶иҜ»еҸ–гҖҒеҶ…вҪ№з«ҜвјқжҺўжөӢгҖҒе‘Ҫд»Өжү§вҫҸгҖҒжӢ’з»қжңҚеҠЎзӯүж”»еҮ»гҖӮ

жјҸжҙһеҺҹеӣ 

еҰӮдёӢжҳҜдёҖдёӘеёёи§Ғзҡ„payload

е®ғжҳҜз”ұдәҺзЁӢеәҸеңЁи§Јжһҗxmlж—¶жңӘзҰҒз”ЁеӨ–йғЁе®һдҪ“еҜјиҮҙзҡ„

и®ҫи®ЎеҲ°зҡ„е…ій”®иҜҚжңү

дҝ®еӨҚж–№жЎҲ

  1. зҰҒз”ЁеӨ–йғЁе®һдҪ“и§Јжһҗ
  1. иҝҮж»Өз”ЁжҲ·жҸҗдәӨзҡ„XMLж•°жҚ®;
  2. дёҚе…Ғи®ёXMLдёӯеҗ«жңүиҮӘе·ұе®ҡд№үзҡ„DTD;

е®Ўи®ЎжҠҖе·§

  1. XML и§ЈжһҗвјҖиҲ¬еңЁй…ҚзҪ®вҪӮд»¶еҜјвјҠгҖҒж•°жҚ®дј иҫ“зӯүжҺҘвјқеҸҜиғҪдјҡж¶үеҸҠпјҢеҮЎжҳҜж¶үеҸҠеҲ° XML и§Ј жһҗзҡ„еңәжҷҜдёӯXML и§ЈжһҗеҷЁжҳҜеҗҰзҰҒвҪӨеӨ–йғЁе®һдҪ“пјҢд»ҺвҪҪеҲӨж–ӯжҳҜеҗҰеӯҳеңЁ XXEгҖӮ

  2. и§Јжһҗе…¶е®ғзү№ж®ҠвҪӮд»¶еңәжҷҜд№ҹеҸҜиғҪеӯҳеңЁ XXE жјҸжҙһпјҢвҪҗеҰӮеңЁзәҝжү“ејҖ Office вҪӮд»¶пјҢдёҠдј  Office вҪӮд»¶зӯүгҖӮ

  3. д№ҹеҸҜж №жҚ®йЎ№вҪ¬еҜјвјҠзҡ„ JAR еҢ…пјҢеҶҚе…ЁеұҖжҗңзҙўеҜ№еә” XML и§ЈжһҗеҢ…дёӯзҡ„е…ій”® APIпјҢжүҫеҲ°еҜ№еә” зҡ„дёҡеҠЎд»Јз ҒиҝӣвҫҸе®Ўи®ЎгҖӮ

SSRF

SSRFпјҲServer-Side Request ForgeryпјҢжңҚеҠЎеҷЁз«ҜиҜ·жұӮдјӘйҖ пјү жҳҜжҢҮж”»еҮ»иҖ…еҲ©з”ЁжңҚеҠЎз«ҜеҠҹиғҪпјҲеҰӮиҺ·еҸ–иҝңзЁӢеӣҫзүҮгҖҒи°ғз”ЁAPIгҖҒзҲ¬еҸ–зҪ‘йЎөзӯүпјүеҸ‘иө·жҒ¶ж„ҸиҜ·жұӮпјҢи®ҝй—®еҶ…зҪ‘иө„жәҗгҖҒжң¬ең°ж–Үд»¶жҲ–дә‘е…ғж•°жҚ®жңҚеҠЎгҖӮ

жјҸжҙһеҺҹеӣ 

дё»иҰҒиҝҳжҳҜз”ұдәҺжңӘеҜ№иҫ“е…ҘurlеҒҡж ЎйӘҢеҜјиҮҙпјҢй…ҚеҗҲдёҖдәӣеҚҸи®®жқҘеҜ№еҶ…зҪ‘иҝӣиЎҢж”»еҮ»пјҢеңЁjavaдёӯпјҢвјҖиҲ¬еҲ©вҪӨhttp/httpsеҚҸи®®жқҘжҺўжөӢз«ҜвјқгҖҒжҡҙвј’з©·дёҫзӯүпјҢиҝҳеҸҜд»ҘеҲ©вҪӨfileеҚҸи®®иҜ»еҸ–/дёӢиҪҪд»»ж„ҸвҪӮд»¶зӯү
